Scarse Project: Adjusting Your Monitor: Pretty Color Targets

Color calibration is really impossible to do without some standard hard-copy color chart reference or densitometer, so this page is completely useless, and has purely decorative value :). If you are looking to do subjective calibration, you will probably be better served by Reference Images page.

Primary colors should appear clean and clear, grays neutral, and transitions between colors in hue charts continuous and really smooth. If there is banding, you are in low-color mode. Also, take a look at grayscale gradients if you haven't already done so.
