Scarse comparison images

(Updated 02/11/1999: images generated using Scarse 0.2-alpha)

Here are some examples of images calibrated using Scarse compared to default output of the driver. All the images are scanned from 35mm transparency film using Polaroid SprintScan 4000. The files are in TIFF format and are about 0.5Mb in size each. JPEG compression was not used because it introduces color artifacts.

All the photos except Kodak Q60 target are mine, and are copyrighted, so please don't steal them... If you want to see more of my work, or to license my images, feel free to visit my photo pages.
